## 下载

### Docker

```
docker pull apachecn0/geekdoc-arch-zh
docker run -tid -p <port>:80 apachecn0/geekdoc-arch-zh
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g geekdoc-arch-zh
geekdoc-arch-zh <port>
# 访问 http://localhost:{port} 查看文档
```
